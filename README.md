# BodyBuddy
가나디와 고냐니가 나오는 체중관리 웹 어플리케이션입니다.
## 멤버구성
 - 강지민
 - 차언주
 - 최지웅
## 기술 스택
Python, CSS, HTML
## 개발 환경
pip install flask
## 실행 방법
git clone https://github.com/djswn/BodyBuddy.git
cd bodyBuddy
python app.py
http://127.0.0.1:5000 접속
## 주요 기능
1. 회원가입, 로그인 기능
2. 사용자 정보 기록
3. BMI, BMR 계산
4. AI 멘트
5. 추천 식단, 추천 운동 제시
6. 캐릭터 화면
